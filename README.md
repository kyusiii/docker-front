# Michelin

## Requirements

- Node 16 / 18

## Install

```npm install```

## Dev / Local

```npm run start```
=> uses DEV env variables

## Build

```npm run build```
=> uses PROD env variables

Generates the build in the dist folder

## Environment variabvles

### Dev / Local

see ```src/environments/environment.ts```

### Prod

see ```src/environments/environment.prod.ts```
